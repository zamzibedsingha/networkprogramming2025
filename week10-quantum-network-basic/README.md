# Week 10: Quantum-Inspired Networking (เครือข่ายจำลองแนวคิดควอนตัม)

## 📌 Overview (ภาพรวมของโปรเจกต์)
Week 10 เป็นการนำแนวคิดทางฟิสิกส์ระดับควอนตัม (Quantum Mechanics) มาประยุกต์ใช้ในการออกแบบระบบเครือข่ายที่เน้นความปลอดภัยขั้นสูง โดยจำลองพฤติกรรมของ **No-Cloning Theorem** (ข้อมูลควอนตัมไม่สามารถถูกคัดลอกได้) และ **State Collapse** (การสังเกตการณ์ทำให้สถานะของข้อมูลพังทลาย)

ระบบนี้ออกแบบมาเพื่อแก้ปัญหา Replay Attacks โดยใช้ Token แบบใช้แล้วทิ้ง (Ephemeral Messages) ซึ่งเหมาะสำหรับระบบที่ต้องการความปลอดภัยของข้อมูลระดับสูงสุด เช่น ระบบจำลองการแจกจ่ายกุญแจควอนตัม (Quantum Key Distribution)

## 🛠️ Key Concepts (กลไกการทำงานหลัก)
- **One-Time-Read Token:** ข้อความถูกห่อหุ้มไว้ใน Token ที่อนุญาตให้อ่านได้เพียง "ครั้งเดียว" เท่านั้น
- **State Collapse (การพังทลายของสถานะ):** ทันทีที่ Token ถูกเปิดอ่าน (Access) สถานะภายในจะเปลี่ยนไป หากมีการพยายามเข้าถึงซ้ำ ข้อมูลจะสลายตัวและคืนค่าเป็น `None` ทันที
- **Token Expiry:** เพิ่มความปลอดภัยด้วยระบบหมดอายุ (TTL - Time to Live) หากไม่มีผู้รับข้อมูลภายในเวลาที่กำหนด Token จะเสื่อมสภาพและใช้งานไม่ได้อีกต่อไป

## 🚀 How to Run (วิธีทดสอบระบบ)

2. รัน Node ด้วยคำสั่ง `python node.py` ในทั้งสองหน้าต่าง
3. ระบบจะทำการแลกเปลี่ยน Quantum Token หากันโดยอัตโนมัติ
